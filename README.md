# Get-It-Done
A temporary to do list website using HTML
you can (start) by typing your tasks then press (+) to add it.
you can check the task by pressing on it. when you are done you can click (finish) and a page will show up telling you if you have finished all your tasks or not. you can also (restart) or (go back) to your own tasks if you clicked (finish) by mistake
! once you leave the website nothing will be saved
